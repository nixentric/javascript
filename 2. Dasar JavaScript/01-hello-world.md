## "script" Tag

Program JavaScript dapat disisipkan hampir di mana saja di dalam dokumen HTML dengan menggunakan tag ``<script>.

Contohnya:

```html
<!DOCTYPE HTML>
<html>

<body>

  <p>Before the script...</p>

  <script>
    alert( 'Hello, world!' );
  </script>

  <p>...After the script.</p>

</body>

</html>
```
